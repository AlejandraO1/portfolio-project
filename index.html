Recent Projects

World Clock App

I created a world clock application using HTML, CSS, JavaScript as well as using the library Moment.js. Utlizing the Moment.js library, I was able to properly format the date and time while taking into account the different time zones. The app will also update the time of each respective city every second without the user having to refresh.

Weather App

I created a weather application using HTML, CSS, JavaScript as well as using a weather API. Within the app, one can search up any place in the world and see the current weather as well as the forecast. Depending on the current weather condition of a city, the background as well as the Spotify music player will change in accordance to the city's current weather condition.
