Japan Landing Page

I created a travel landing page for Japan using HTML and CSS. I embedded videos from YouTube as video itineraries that further talks about traveling to Japanese cities. There is a consistent aspect ratio for the videos no matter how the window is resized. I made this landing page responsive without the usage of any frameworks.

World Clock App

I created a world clock application using HTML, CSS, JavaScript as well as using the library Moment.js. Utilizing the Moment.js library, I was able to properly format the date and time while taking into account the different time zones. The app will also update the time of each respective city every second without the user having to refresh the page.
